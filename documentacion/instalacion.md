# Instalación del Proyecto Memes

## Requisitos
- Java JDK 17 o superior
- Terminal o consola de comandos
- Sistema operativo compatible con Java (Windows, Linux o MacOS)

## Descarga del proyecto
1. Descargar el repositorio del proyecto.
2. Descomprimir el archivo en una carpeta de tu ordenador.

## Compilación del programa
Abrir una terminal dentro de la carpeta `src` del proyecto y ejecutar:

javac *.java

Esto generará los archivos `.class` necesarios.

## Ejecución del programa
Para ejecutar el programa:

java Main

## Desinstalación
Para desinstalar el programa basta con eliminar la carpeta del proyecto.
